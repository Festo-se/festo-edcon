CLI

festo-edcon

position

This is used to start a position task on a drive determined by the IP address and the respective parameters of the position task.

pnu

This tool can be used to access PNUs of a drive

festo-edcon-gui

Starts the graphical user interface where the user can easily start motion jobs, inspect and manipulate process data, observe the PROFIdrive state machine and read/write parameters.
